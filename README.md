# Adult Census Income Classification

## Overview
This project predicts whether an individual's annual income exceeds $50,000 using the Adult Census Income dataset.

## Objectives
- Understand the dataset
- Perform data cleaning
- Apply feature engineering
- Train multiple classification models
- Evaluate model performance

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
